# Minesweeper - 经典扫雷小游戏（Jetpack Compose版）

一款使用 **Jetpack Compose** 开发的 Android 扫雷小游戏，复刻经典 Windows 扫雷玩法，适合手机触屏操作。



## 游戏特色

- 经典扫雷玩法：首次点击必安全，自动展开空白区域
- 长按标记旗帜，单击翻开格子
- 实时显示剩余雷数与游戏用时
- 胜利/失败状态提示
- 响应式布局，适配不同屏幕尺寸
- 简洁的 Material Design 3 风格

## 游戏规则

- **目标**：翻开所有非雷格子，不点到地雷
- **操作**：
  - **单击** → 翻开格子
  - **长按** → 标记/取消旗帜（表示怀疑有雷）
- **数字含义**：周围 8 个格子中地雷的数量
- **空白格**：自动连锁翻开周围安全区域
- **胜利**：翻开所有非雷格子
- **失败**：点到地雷

## 技术栈

- **语言**：Kotlin
- **UI 框架**：Jetpack Compose
- **最低 API**：Android 8.0 (API 26)
- **架构**：纯 Compose + State Hoisting + remember

## 当前版本参数

- 网格：10 × 10
- 地雷数量：15 个
- 难度：初级（适合手机操作）
